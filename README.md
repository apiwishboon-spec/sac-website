# Suankularb Astronomy Club (SAC) Website

Official website for the Suankularb Astronomy Club (SAC) at Suankularb Wittayalai School.

## 🌟 About

The Suankularb Astronomy Club is dedicated to exploring the wonders of the universe, fostering astronomical education, and inspiring the next generation of astronomers and space explorers at the heart of Suankularb.

## 🚀 Features

- **Modern Responsive Design**: Built with HTML5, CSS3, and JavaScript
- **E-commerce Store**: SAC merchandise shop with secure checkout
- **News & Updates**: Latest astronomy news and club announcements
- **Planetarium Information**: Details about the Sirindhorn Planetarium
- **Club Administration**: Organizational structure and team information
- **Order Management**: Complete order processing with email notifications
- **Mobile Friendly**: Fully responsive design for all devices

## 🛍️ Shop Features

- Product catalog with images and descriptions
- Shopping cart functionality
- Secure checkout with OTP verification
- Payment integration with PromptPay QR codes
- Order success tracking
- Email notifications for order status updates

## 📧 Order System

- **Order Confirmation**: Automatic email receipts
- **Status Tracking**: Real-time order status updates
- **Email Notifications**: Automated emails for status changes
- **Payment Processing**: Secure payment slip verification
- **Customer Support**: Contact information and support channels

## 🎨 Design & Technology

- **Frontend**: Pure HTML, CSS, JavaScript (no frameworks)
- **Styling**: Custom CSS with glassmorphism effects
- **Icons**: Font Awesome 6.4.0
- **Backend**: Google Apps Script for order processing
- **Payment**: PromptPay QR code generation
- **Security**: Cloudflare Turnstile protection

## 🛠️ Setup

1. Clone this repository
2. Set up Google Apps Script backend
3. Configure environment variables:
   - `GAS_WEB_APP_URL`: Google Apps Script endpoint
   - Turnstile site key for security
4. Deploy to your preferred hosting platform
